# Система дистанционного обучения сотрудников

Технологии: Vue 3, Vite, Vue Router, Pinia. Данные и прогресс сохраняются в localStorage.

## Запуск

1. Установите зависимости:
```bash
npm install
```
2. Запустите локально:
```bash
npm run dev
```
3. Откройте `http://localhost:5173`.

## Пользователи (моки)
- Сотрудник-методист: `ivan.m / 1234`
- Режиссёр: `olga.r / 1234`
- Поддержка: `petr.s / 1234`
- HR: `hr / hr`

## Структура
- 4 блока: «Знакомство с компанией», «Введение в должность», «Обязательное обучение», «Развитие»
- Для блоков 2–4 контент различается по должностям
- Статусы: «не пройдено», «пройдено», «проверено»
- После теста (>80%) статус блока меняется на «пройдено», HR может вручную менять статус и дедлайны

## Страницы
- Личный кабинет сотрудника: список блоков, статусы, дедлайны
- Страница блока: темы (ссылки/страницы), ссылка на тест
- Тест: простой пример, фиксированные вопросы
- HR-панель: список сотрудников, следующий дедлайн
- HR-профиль сотрудника: установка дедлайнов и изменение статусов
